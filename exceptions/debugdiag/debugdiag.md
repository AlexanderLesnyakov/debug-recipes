### dbghost.exe

It's a tool that provides COM based API for dbgeng. A great example of usage was written by Naveen in the post: <http://naveensrinivasan.com/2010/11/11/using-managed-code-to-debug-memory-dumps/> where he presents how to issue parallel `GCRoot` command using those debug interfaces.
